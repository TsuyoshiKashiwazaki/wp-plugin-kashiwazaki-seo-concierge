# Changelog

本プロジェクトの主な変更点を記録します。フォーマットは [Keep a Changelog](https://keepachangelog.com/ja/1.0.0/) に準拠し、[セマンティック バージョニング](https://semver.org/lang/ja/) を採用しています。

## [1.0.1] - 2026-06-30

### Added

- リンク到達性チェック。索引したページが実際に開けるか（HTTP ステータス）を定期的に確認し、見つからない（404）・エラー（403・500・タイムアウト等）のページを回答候補から自動で外します。復旧すると自動で戻ります。一時的な不調で正常なページを誤って外さないよう、404・410 は即時、その他のエラーは2回続けて失敗したときに外します。管理画面「インデックス」タブに状態の内訳とリンク切れ一覧、手動チェックボタンを追加。
- スパム対策のための IP アドレス・ブラウザ（ユーザーエージェント）記録（任意・既定で有効）。質問ログに記録し、保存日数の経過で自動削除します。Cloudflare 経由のサイト向けに、実際の訪問者 IP を正しく判定する設定を追加（CF の公開 IP レンジ照合つき）。これにより連投制限も訪問者単位で正しく機能します。汎用リバースプロキシ向けの信頼プロキシ設定も追加。
- チャット会話のタブ内保持。ページの再読み込みや遷移をしても、タブを閉じるまで会話が残るようになりました（ブラウザの sessionStorage を使用）。

### Changed

- 管理画面「インデックス」タブのレイアウトを整理。「状態」と「設定」を分け、長い説明はツールチップにまとめ、見やすくしました。

### Fixed

- 回答キャッシュの整合性。ページが除外・リンク切れ等になった際、そのページを含む古い回答キャッシュを自動で削除し、削除済みページが再び案内される問題を防ぎます。
- チャットで返信を待たずに連続送信すると、吹き出しの表示順が崩れることがある問題を修正（送信中は次の送信を一時的に無効化）。

## [1.0.0] - 2026-06-29

### Added

- 初回リリース。
- サイト内 AI コンシェルジュ（フローティングチャット）。訪問者の質問に対し、サイト内の最適なページを意味的に検索して案内します。
- sitemap.xml / llms.txt の解析によるページ索引化（カスタムテーブル + WP-Cron 差分更新、手動再構築、除外ルール）。
- 検索 AI（埋め込み）と回答 AI（チャット）の役割分担。プロバイダ／モデルはそれぞれ差し替え可能。
- コスト上限（1 日・1 か月）、リクエスト回数上限、レート制限、トークン上限などの安全装置。
- API キーの暗号化保存（フロント非出力）、個人情報の best-effort マスク、ログ保存日数、同意取得などのプライバシー機能。
- 表示カスタマイズ（ボタン文言・チャットタイトル・回答アイコン画像・テーマカラー・あいさつ・質問例・表示ページ）。スマートフォンではアイコンボタンに自動切替。
- 分析機能（訪問者／管理者テストの分離集計、人気の質問、コンテンツギャップ、案内ページ付き履歴、CSV 出力）。
- 任意の GA4 イベント連携。
- 利用者向け HTML マニュアル（`docs/`）。
